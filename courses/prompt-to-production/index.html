The Project

Learn by building a real SaaS product.

We don't teach abstract theory. Over 4 weeks, you will build and launch PocketBudget—a secure, cloud-connected expense tracker with a real database and payment integrations.

Cloud Database (Supabase)

Set up secure user authentication and a Postgres database. Learn to protect user data with Row Level Security (RLS).

Cross-Platform Sync

Build a web app (Next.js) and a mobile app (Expo/React Native) that talk to the exact same backend in real-time.

Live Funding & Deployment

Integrate Paystack for live web wallet funding. Deploy to Vercel and compile binaries for the iOS and Google Play stores.

System Setup & Tools Integration
